**Google SEO Query Anomaly Detection**

    Demo - https://huggingface.co/spaces/aakash-malhan/seo-search-anomaly-detection 

**Business Problem**

    Websites with thousands of ranking keywords often lose traffic silently due to drops in CTR, impressions, or ranking position.
    Manual monitoring does not scale, traditional dashboards miss query-level anomalies.


**We solve**

    Which search queries are behaving abnormally and need SEO attention right now.
    Allows proactive SEO + content decisions before traffic loss compounds.
    Works daily / weekly as automated quality-control for search performance.
